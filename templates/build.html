インスタンス(YUKIYOUTUBE)作成方法


全て自己責任です。

Render編

  • １：renderのアカウントを作成(ログイン)

  • ２：blueprintをクリック

  • ３：New Blueprint Instanceをクリック

  • ４：Public Git repositoryの欄にGithubのURLをペースト、Continueをクリック(GithubのURLは一番下にあります)

  • ５：適当な名前を付けてCreate New ResourcesもしくはApplyをクリック

  • ６：Dashboardに戻り作成したものをクリック、左上に表示されているURLが作成したインスタンスです


    • このインスタンスの最新版は環境変数(Environment Variables)でhcaptchaという名前の環境変数を作成してhCaptchaで取得したサイトキーをコピペして下さい
    https://hCaptcha.com/?r=9f4c780b7065

    codesandbox.io編

  • 1.アカウント作成

  • 2.左上の「+repository」をクリック

  • 3.「Find by URL」をクリックして下のGithubのURLを入力して下に表示されたものをクリック

  • 4.画面が変わったら暫くして「Enable VS Code Web」をクリック

  • 5.キーボードがあるならCTRL(コントロールキー)+SHIFT(シフト)+C、タブレットは左上のメニューみたいなボタンを押して「Terminal」を押して「New Terminal」を押す

  • 6.下に出てきた入力欄に「pip install -r requirements.txt&uvicorn main:app --reload」を入力してエンター
